- abrir dois terminais
- no primeiro rodar o comando que inicia a aplicação em ejs
    npm run dev 
- no segundo  se já criou a venv antes, basta iniciar a venv:
    source venv/bin/activate
- após iniciar a API
     python3 ApiTradu.py

    python3 -m venv venv
